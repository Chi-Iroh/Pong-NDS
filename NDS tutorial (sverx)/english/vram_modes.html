Note:

(1): These synonyms can be misleading as they configure the memory bank at addresses that are not at the beginning of the background/sprite memory. As much as possible, avoid using these synonyms to prevent issues.

(2): When the memory bank is configured as a texture palette or extended palette for backgrounds/sprites, the memory is not accessible by the processor. Therefore, to read/write within these banks, it will be necessary to temporarily change modes.

(3): Backgrounds 0 and 1 can use the extended palettes of backgrounds 2 and 3, respectively, if properly configured.
